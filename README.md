## Installation

POS requires [Node.js](https://nodejs.org/) v20+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd final-report
npm i
npm start
```

For production environments...

```sh
npm install --production
npm production
```

Ready to use in your browers:

```sh
localhost:3334/
```
